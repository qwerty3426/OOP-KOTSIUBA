# Лабораторна робота №28
## Тема: Серіалізація об’єктів у JSON

### Виконавець: [Твоє Прізвище та Ім'я]
### Варіант: №5 (Музика: Song, Artist, SongRepository)

---

## Опис проєкту
Проєкт демонструє роботу з бібліотекою `System.Text.Json` для серіалізації та десеріалізації об'єктів у мові програмування C#. 

**Основні можливості:**
* Створення об'єктів класів `Song` (Пісня) та `Artist` (Виконавець).
* Асинхронне збереження списку пісень у файл `songs.json`.
* Асинхронне завантаження даних із JSON-файлу.
* Пошук об'єктів за ID у репозиторії.

## Структура програми
* **Models/** — класи предметної області (`Artist.cs`, `Song.cs`).
* **Repositories/** — клас `SongRepository.cs`, що реалізує логіку роботи з даними та JSON.
* **Program.cs** — точка входу, де демонструється виконання всіх пунктів завдання.
